# Nova Dash 🚀
## Overview
NovaDash is a space-themed vertical shooter game built using [p5.js](https://p5js.org/). It was created for **Excel 2024**, the 25th edition of the awesome techno-managerial fest hosted by Govt. Model Engineering College, Kochi. With over 600 concurrent players giving it a shot, NovaDash turned out to be a crowd favorite at the event! 🚀
<br />
 
## Preview
![novadash.png](public/images/novadash.jpg)
<br />
 
## 🪐 Features
- Fast-paced gameplay that keeps you hooked.  
- A leaderboard to show off your high scores.  
- Works smoothly on different devices.  
- Fun space visuals and vibes. 
<br />
## ⚙️ Technical Architecture
- Engineered the full rendering pipeline for a web-based 2D vertical space-shooter, successfully supporting **600+ concurrent players**.
- Implemented **deterministic rigid-body physics** and collision detection to ensure consistent, reproducible gameplay across sessions and devices.
- Optimized complex spatial queries using **Axis-Aligned Bounding Box (AABB) Collision Detection**, guaranteeing a stable **60 FPS** during gameplay.
- Built a **Dynamic Difficulty Scaling** system that adapts enemy spawn rate and pacing in real time based on player performance.
<br />
